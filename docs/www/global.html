(constant) InstructionState

An instruction ends when it returns truthy or it reached the end. The output value must be assigned to the out variable within the scope.
Properties:
Name Type Description
fns Array.<function()> An ordered instructions list.
nextFn int Next instruction within the instruction set.
scope Object scope of the instruction set.
Properties
Name Type Description
out any The value returned should be assigned to this property. Everything will be accepted as an output variable. Even 'undefined'.
err Error The last error which occured. If the error is not assigned a value 'handled = true' the error will be passed to the the instruction will be ended if the error is not marked as handled. And then passed to the previous instruction and so forth.

Example
// First set1 calls set2 which at some point returns a value
// The value unless its undefined will be passed as argument
// to the next instruction in set1 (which is #1)
