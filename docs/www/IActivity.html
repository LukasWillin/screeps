Interface: IActivity

IActivity

An activity implements the logic to execute specific task(s). The activity class never creates instances but implements the static instructions to execute upon an activity entity.

(static) run :IInstruction

An activity must at least implement one instruction to be run by the scheduler with the property name 'run'. This is the root instruction at which the execution starts or resumes from.
