Interface: IActivity

activity.IActivity

An activity implements the logic to execute specific task(s). The activity class never creates instances but implements the static instructions to execute upon an activity entity.

Methods

(static) initCache(activityEntity, nextInstruction, cache)

Initializes given cache object with required data to run a certain instruction. The init cache is ran once before the first call to any instruction.
Parameters:
Name Type Description
activityEntity IActivityEntity The persisted activity entity from memory.
nextInstruction IInstruction The instruction which will be called next.
cache Object An empty object to initialize. This object is a shortcut to 'activityInstance.cache'.
