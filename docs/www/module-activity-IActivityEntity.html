Interface: IActivityEntity

activity~IActivityEntity

The activity entity contains all info about an active activity. Note that the only information which should be accessible to instruction functions is the memory (mem) property.
Properties:
Name Type Description
type string Registered type (class) of the activity.
mem Object The memory of the activity. Which is also served as the memory (mem) argument for instruction functions.
Properties
Name Type Description
cache Object A cache object which is made available on the memory object. This object must be removed before it is stored in memory or be marked as not 'enumerable'.
stats Object Statistics about its resource usage. Here the kernel stores and tracks the ressource usage of this activity
execStack Array.<IInstructionExecutionState> The execution stack in order to be able to resume it later.
